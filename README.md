# 📊 マイ株価ダッシュボード

日本株の株価・ファンダメンタル指標・ニュースをまとめて確認できる Streamlit 製の個人向け株式管理ダッシュボードです。

---

## 🖼️ 機能概要

### 📊 サマリーダッシュボード
登録中のすべての銘柄を一覧表示します。

- 現在値・前日比（上昇 🟢 / 下落 🔴 の色分け表示）
- 時価総額 / PER / PBR / 配当利回り
- 最終更新日時の表示

### 🔍 銘柄詳細ページ
個別銘柄の詳細情報を確認できます。

- **株価チャート** — 1ヶ月〜5年の期間を選択可能
- **ファンダメンタル指標** — 時価総額・PER（実績/予想）・PBR・EPS・ROE・配当利回り・52週高値/安値・業種・セクターなど
- **最新ニュース** — Google News RSS から自動取得（日本語）
- **会社HP** — アプリ内に iframe で表示

### ⚙️ 銘柄管理
- **追加**: 表示名とティッカーシンボルを入力して登録
- **削除**: 登録済み銘柄をサイドバーから削除
- データは SQLite（`stocks.db`）に永続保存
- デフォルトでイオン（8267.T）・ソニーグループ（6758.T）が登録済み

### 🌙 ダークモード
サイドバーのトグルでダーク/ライトを切り替え可能

## 🚀 セットアップ・起動方法

### 1. 依存パッケージのインストール

```bash
pip install -r requirements.txt
```

**requirements.txt の内容例:**

```
streamlit
yfinance
pandas
feedparser
```

### 2. アプリの起動

```bash
streamlit run app.py
```

ブラウザで `http://localhost:8501` が自動的に開きます。

## 🔧 使い方

### 銘柄の追加
1. サイドバーの「➕ 銘柄を追加」セクションへ
2. **表示名**（例: `トヨタ自動車 (7203)`）を入力
3. **ティッカー**（例: `7203.T`）を入力
4. 「追加」ボタンをクリック

> 💡 東証銘柄のティッカーは `証券コード.T` の形式です（例: `7203.T`）。

### 銘柄の削除
1. サイドバーの「🗑️ 銘柄を削除」セクションへ
2. 削除したい銘柄を選択して「削除」ボタンをクリック

### 期間の変更（詳細ページ）
サイドバーの「期間」セレクトボックスから選択:

| 表示 | 期間 |
|------|------|
| 1mo  | 1ヶ月 |
| 3mo  | 3ヶ月 |
| 6mo  | 6ヶ月 |
| 1y   | 1年（デフォルト）|
| 2y   | 2年 |
| 5y   | 5年 |

## ⚠️ 注意事項

- 株価データは [yfinance](https://github.com/ranaroussi/yfinance) 経由で Yahoo Finance から取得しています。市場の状況によっては取得に失敗する場合があります。
- 会社HP の iframe 表示は、サイト側の設定（`X-Frame-Options` など）によって表示できない場合があります。
- 本アプリは個人利用・学習目的を想定しています。投資判断にはご自身の責任でご利用ください。
